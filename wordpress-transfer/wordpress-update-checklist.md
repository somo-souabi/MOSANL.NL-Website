# WordPress Update Checklist

Gebruik deze checklist voor een handmatige update van de live WordPress-website. Deze repository is niet gekoppeld aan de live website; publiceer niets automatisch vanuit GitHub.

## Voorbereiding

- [ ] Maak eerst een volledige backup van WordPress: bestanden, database, thema-instellingen en uploads.
- [ ] Controleer of u toegang heeft tot `wp-admin`.
- [ ] Open de live website in een apart browservenster om de huidige situatie te vergelijken.
- [ ] Download of noteer bestaande paginacontent voordat u iets overschrijft.

## Afbeeldingen uploaden

- [ ] Upload `images/Logo.png` naar de WordPress Media Library.
- [ ] Upload `images/Festival rotterdam.png` naar de WordPress Media Library.
- [ ] Upload `images/mustapha 1.png` naar de WordPress Media Library.
- [ ] Upload `images/Mustapha 2.png` naar de WordPress Media Library.
- [ ] Upload `images/Mustapha 3.png` naar de WordPress Media Library.
- [ ] Upload `images/Mustapha 4.png` naar de WordPress Media Library.
- [ ] Vul alt-teksten in volgens `wordpress-image-map.md`.
- [ ] Vervang oude afbeeldingsverwijzingen op pagina's door de nieuwe Media Library-afbeeldingen.

## Pagina's bijwerken

- [ ] Open `wp-admin`.
- [ ] Update de homepage met `home-page-content.md`.
- [ ] Update de pagina Over MOSA met `over-mosa-page-content.md`.
- [ ] Update de pagina Diensten met `diensten-page-content.md`.
- [ ] Update de pagina Activiteiten met `activiteiten-page-content.md`.
- [ ] Update de pagina Nieuws met `nieuws-page-content.md`.
- [ ] Update de pagina Contact met `contact-page-content.md`.
- [ ] Controleer dat er geen placeholdertekst live blijft staan tenzij bewust gewenst.

## Navigatie, knoppen en formulieren

- [ ] Controleer het menu: Home, Over MOSA, Diensten, Activiteiten, Nieuws, Contact.
- [ ] Controleer alle knoppen en links.
- [ ] Controleer het contactformulier.
- [ ] Voeg alleen bevestigde e-mailadressen, telefoonnummers en adressen toe.
- [ ] Controleer dat sfeerbeelden niet worden gepresenteerd als echte bestuursleden, medewerkers, partners of testimonials zonder bevestiging.

## Styling en responsive controle

- [ ] Voeg eventuele veilige CSS toe via WordPress Customizer > Additional CSS.
- [ ] Controleer de header op desktop, tablet en mobiel.
- [ ] Controleer de hero-secties op desktop, tablet en mobiel.
- [ ] Controleer kaarten/blokken op voldoende witruimte en leesbaarheid.
- [ ] Controleer de footer op desktop, tablet en mobiel.
- [ ] Controleer mobiele weergave en menuwerking.

## Afronding

- [ ] Clear cache in WordPress-cacheplugin, hostingcache en browsercache.
- [ ] Test de live website in een incognito/private browservenster.
- [ ] Test de live website op mobiel.
- [ ] Controleer of alle afbeeldingen laden.
- [ ] Controleer of alle pagina's vindbaar zijn via het menu.
- [ ] Controleer of de contactpagina werkt.
- [ ] Bewaar deze transfermap als documentatie voor toekomstige updates.
